# Loottable プラグイン 使用方法

## 概要
サーバー上のすべてのルートコンテナ（クレート、NPC死体、採集物など）のドロップ内容をゲーム内UIから編集・管理できる高機能ルートテーブルプラグインです。スタックサイズの変更、リサイクラー設定、製錬速度変更、採集倍率なども統合的に管理可能です。

## 主な機能
- **ルートテーブル編集**: 各クレート・NPC・採集物ごとにドロップアイテム、確率、個数をUI上で編集
- **スタックサイズ変更**: 全アイテムのスタック上限を個別にカスタマイズ
- **リサイクラー設定**: リサイクラーの出力アイテムと倍率を変更
- **サプライドロップ設定**: 物資投下の中身をカスタマイズ
- **製錬速度**: 炉の製錬速度を倍率で変更
- **採集倍率**: 採掘・伐採・拾得アイテムの倍率を変更
- **カスタムプリセット**: ルートプロファイルの保存と切り替え

## パーミッション
| 権限名 | 説明 |
|---|---|
| `loottable.edit` | ルートテーブル編集UIへのアクセス権限 |
| `loottable.test` | テスト用ルート生成の権限 |

## コマンド
| コマンド | 説明 |
|---|---|
| `/loottable` | ルートテーブル編集UIを開く（設定でコマンド名変更可能） |
| `/loottable refresh` | 全コンテナのルートを再生成 |

---

## サーバー管理者向け（ファイル操作）

### プラグインの導入
1. `Loottable.cs` を `carbon/plugins/` に配置します。
2. 設定ファイル: `carbon/config/Loottable.json`
3. データ: `carbon/data/Loottable2/` にルートテーブルデータが保存される
4. リロード: `carbon.reload Loottable`

### 権限の付与
```
o.grant user <SteamID> loottable.edit
```

### 連携プラグイン
- **ImageLibrary**: アイテムアイコンの表示に使用
- **CustomItemDefinitions**: カスタムアイテムの対応

## ゲーム内管理者向け（Carbon CPanel）

### CPanel へのアクセス
- `F1` コンソールで `cpanel` と入力、またはチャットで `/cpanel` を実行します。

### プラグインの管理
1. **「Plugins」** タブから **Loottable** を選択。
2. **Reload** でリロード、**Config** で設定を確認できます。

### ルートテーブルの編集
- `loottable.edit` 権限を持つ管理者は、ゲーム内で `/loottable` を実行するとUIが開きます。
- UIからクレートやNPCを選択し、ドロップアイテムの追加・削除・確率変更が可能です。

### 権限の管理
1. **「Permissions」** タブで `loottable.edit` を管理者に付与します。
