# GameOverScreen プラグイン 使用方法

## 概要
プレイヤーが死亡（ゲームオーバー）した際に、画面全体を覆うUI表示とサウンドエフェクトで死の演出（アニメーション）を追加するシンプルなプラグインです。背景画像の上にオーバーレイ画像（例：YOU DIEDなどのテキスト画像）を重ねて拡大表示させるアニメーション機能があります。

## 設定（Configuration）
`oxide/config/GameOverScreen.json` にて以下の設定が変更可能です。

| 設定項目 | 型 | デフォルト値 | 説明 |
|---|---|---|---|
| `BackgroundImageUrl` | 文字列 | `https://i.imgur.com/RY3DrU6.jpeg` | 背景としてフェードインしてくる画像のURLです。 |
| `OverlayImageUrl` | 文字列 | `https://i.imgur.com/RHw6pie.png` | 背景の上に重なり、徐々に拡大（ズーム）するテキストやロゴの画像URLです。 |
| `OverlayScaleFactor` | 数値 | `1.2` | OverlayImageUrlの最終的な拡大率です。 |
| `AnimationFPS` | 数値 | `60` | UIアニメーションの滑らかさ（更新頻度）です。 |
| `SoundEnabled` | 真偽値 | `true` | UI表示と共に専用のミッション失敗サウンドを再生するかどうか。 |

## 主な仕様
- **動作条件**: プレイヤーがNPCゾンビ以外の要因で死亡した際に(`OnPlayerDeath`フック)、自動的に演出が開始されます。
- **表示時間**: フェードイン（約0.5秒）→ 表示維持（約5秒）→ フェードアウト（約1秒）のサイクルで画面に画像が表示されます。
- **権限・コマンド**: このプラグインには特定の権限（パーミッション）やチャットコマンドは用意されておらず、単に導入して動作させるバックグラウンドタイプのプラグインです。
- **依存環境**: 画像のダウンロード・キャッシュ機能のために `ImageLibrary` プラグインが必須です。
