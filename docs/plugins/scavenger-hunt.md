# ScavengerHunt プラグイン 使用方法

## 概要
サーバー内にランダムに宝箱（クレート）を出現させ、プレイヤー達にそれを見つけさせる「宝探し（スカベンジャーハント）」イベントを自動または手動で開催するプラグインです。
イベント中、プレイヤーの画面には宝箱の方向と距離を示す専用のHUD（コンパス）が表示され、一番最初に見つけて開けた人が報酬（アイテムやポイント）を獲得できます。

## パーミッション・コマンド一覧
一般プレイヤーが使用するコマンドはありません。以下のコマンドは管理者（Admin）専用です。

| コマンド | 説明 |
|---|---|
| `/sch_start` | （チャットコマンド）ScavengerHuntイベントを即座に開始します。 |
| `/sch_stop` | （チャットコマンド）現在進行中のScavengerHuntイベントを強制終了させ、宝箱を消去します。 |

## 主な機能と特徴
- **自動化対応**: 設定した一定時間ごとに自動でイベントが開催されます。
- **UI表示（コンパス）**: イベント開始時、プレイヤーのHUDに宝箱の方角を示すインジケーター（N, S, W, Eなど）と、宝箱までの距離が表示されます。
- **ルート連携**: 
  - イベント専用のドロップ内容を独自設定できるほか、`SimpleLootTable` プラグインと連携して既存のルートテーブル名を指定することも可能です。
- **ポイント連携**: 
  - `ServerRewards` プラグインが導入されている場合、宝箱を一番に見つけたプレイヤーに設定したポイント（RewardPoints）を付与できます。
- **通知連携**: `Notify` プラグインによるチャットの装飾表示に対応。

## 設定（Config）について
`oxide/config/ScavengerHunt.json` から詳細な設定が可能です。（一部抜粋）
- `Autostart event`: `true` にすると自動開催が有効になります。
- `Pre-event message time`: イベント開始を事前に（例：60秒前）チャットで予告します。
- `Minimum amount of online players`: この人数以上のプレイヤーが居ないとイベントは自動開始されません。
- `Own loot table for crate`: `true` にするとプラグイン独自のドロップアイテムリスト（アイテム、確率、個数）が使用されます。
