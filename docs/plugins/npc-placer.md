# NpcPlacer プラグイン 操作マニュアル

BetterNpc のカスタムスポーンポイント設定を、ゲーム内チャットコマンドで視覚的に行える管理者向けツールです。

---

## セットアップ

### 1. プラグインの配置

`NpcPlacer.cs` をサーバーの `oxide/plugins/` フォルダに配置します。

### 2. パーミッションの付与

このプラグインは **管理者（Admin）** または **`npcplacer.admin` パーミッション保持者** のみ使用できます。

```
o.grant user <SteamID> npcplacer.admin
o.grant group admin npcplacer.admin
```

---

## 基本的な使い方

### ステップ1: スポーンポイントを新規作成する

NPCを配置したい場所の中心に立ち、以下のコマンドを実行します。

```
/npc new "INDUSTRIAL CARGO"
```

- プレイヤーの足元が**中心座標**として登録されます
- 空のスポーンポイント設定ファイルが `oxide/data/BetterNpc/Custom/INDUSTRIAL CARGO.json` に作成されます
- 自動的に**編集モード**に入ります

### ステップ2: NPCの出現位置を追加する

編集モードに入ると、画面上部にステータスバーが表示されます。

#### 足元に追加

NPCを湧かせたい場所に移動し、以下を実行します。

```
/npc add
```

#### エイム先に追加（遠距離）

高い場所やアクセスしにくい場所には、その地点を見ながら以下を実行します。

```
/npc aim
```

- 追加された座標は**緑色の球体マーカー**と**番号**でリアルタイム表示されます
- 中心座標は**赤色の球体マーカー**で常時表示されます

### ステップ3: 設定を調整する

```
/npc config
```

設定画面（CUI）が開きます。以下の項目を調整できます。

| 項目 | 説明 |
|---|---|
| 有効 | このプリセットのON/OFF |
| NpcSpawnプリセット名 | NpcSpawnプラグインのプリセット名（例: `Oil-Rig-Worker-Blue`） |
| 昼の最小数 / 最大数 | 昼間の出現NPC数 |
| 夜の最小数 / 最大数 | 夜間の出現NPC数 |
| リスポーン最小/最大(秒) | 死亡後のリスポーン時間 |
| スポーンタイプ | `Custom`（座標指定）または `Random`（ランダム） |
| 全ポイントにスポーン | ONにすると全座標にNPCが配置されます |
| モニュメントサイズ | 判定範囲 (X, Y, Z) |

### ステップ4: 保存して編集終了

```
/npc save
/npc edit
```

`/npc save` で JSON ファイルに保存され、`/npc edit`（引数なし）で編集モードを終了します。

### ステップ5: BetterNpc に反映

サーバーコンソールで以下を実行して BetterNpc をリロードします。

```
o.reload BetterNpc
```

## 複数プリセットの運用

1つのスポーンポイントに**複数のNPC種類**を配置する場合の手順です。

### 例: INDUSTRIAL CARGO に2種類のNPCを配置

1. `/npc edit "INDUSTRIAL CARGO"` で編集モード開始
2. `/npc config` で設定画面を開く
3. Preset #1 の設定を行う（例: `Oil-Rig-Worker-Blue`, 昼21体）
4. 画面上部の `+` ボタンで Preset #2 を追加
5. `#2` タブをクリックして切り替え
6. Preset #2 の設定を行う（例: `Older-Oil-Rig-Worker-Blue`, 昼5-7体）
7. 「保存」ボタンで保存
8. 設定画面を閉じる

プリセットごとに異なるスポーン座標を設定する場合:
1. `/npc preset 1` でプリセット#1に切り替え
2. `/npc add` や `/npc aim` で座標を追加
3. `/npc preset 2` でプリセット#2に切り替え
4. 別の座標を `/npc add` で追加

## コマンド一覧

| コマンド | 説明 | 編集モード |
|---|---|---|
| `/npc edit` | 最寄りスポーンを自動検出して編集開始/終了 | 不要 |
| `/npc edit "名前"` | 指定スポーンの編集開始 | 不要 |
| `/npc new "名前"` | 新規スポーン作成（現在地が中心） | 不要 |
| `/npc add` | 足元座標を追加 | 必要 |
| `/npc aim` | エイム先座標を追加 | 必要 |
| `/npc remove 番号` | 指定番号の座標を削除 | 必要 |
| `/npc clear` | 全座標をクリア | 必要 |
| `/npc preset 番号` | プリセット切り替え | 必要 |
| `/npc save` | JSONファイルに保存 | 必要 |
| `/npc config` | NPC設定画面（CUI）を開く | 必要 |
| `/npc list` | カスタムスポーン一覧表示 | 不要 |
| `/npc tp "名前"` | スポーン中心にテレポート | 不要 |

## データファイルの場所

```
oxide/data/BetterNpc/Custom/
├── INDUSTRIAL CARGO.json
├── COAST CASINO.json
├── Oil Rig 4.json
└── ...
```

NpcPlacer が作成・編集するファイルは全て BetterNpc の読み込み対象と同じフォルダに保存されます。ファイルの共有により、NpcPlacer で設定した内容が BetterNpc のリロード時にそのまま反映されます。

## よくある質問

**Q: `/npc edit` で「付近にカスタムスポーンポイントが見つかりません」と出る**
A: まだカスタムスポーンが作成されていません。`/npc new "名前"` で新規作成してください。

**Q: 座標を追加しても NPC が湧かない**
A: `/npc save` で保存した後、サーバーコンソールで `o.reload BetterNpc` を実行してください。

**Q: マーカーが消えてしまった**
A: マーカーは5秒ごとに自動再描画されます。少し待つと再表示されます。

**Q: 設定画面で数字を入力するとツールバーが反応する**
A: 最新版では修正済みです。`KeyboardEnabled` により入力中はゲーム操作がブロックされます。

**Q: パーミッションはどう設定する？**
A: サーバーコンソールで `o.grant user <SteamID> npcplacer.admin` を実行してください。サーバー管理者（Admin権限）は自動的に使用できます。
