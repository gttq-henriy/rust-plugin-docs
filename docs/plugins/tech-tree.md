# TechTree プラグイン 使用方法

## 概要
Rustの標準機能である「テックツリー（技術ツリー）」を完全にカスタマイズし、独自のアイテムアンロック経路や価格設定を可能にするプラグインです。直感的なゲーム内UIを備え、管理者はゲーム内で直接ツリーを編集できます。

## パーミッション（権限）
プレイヤーやグループに対して、以下の権限（パーミッション）を付与することで動作を制御します。

| 権限名 | 説明 |
|---|---|
| `techtree.use` | プラグインによってカスタマイズされたテックツリーを利用するための基本権限です。通常のプレイヤーに付与します。 |
| `techtree.unlock.free` | スクラップなどの通貨を消費せずに、無料でアイテムをアンロックできる権限です。（VIP向け等に活用） |
| `techtree.unlock.instant` | アンロックにかかる時間をゼロにし、即座に完了させる権限です。 |
| `techtree.unlock.path` | ツリーの前提ルート（パス）を無視して、好きなノード（アイテム）を直接アンロックできる権限です。 |

## 主な機能と操作方法

### プレイヤーの基本操作
1. **ワークベンチ（レベル1～3）に近づく**。
2. 標準の操作キー（リロードキー `R` など）またはUIメニューから **「Open tech tree」** などのボタンを実行。
3. プラグイン専用のカスタムUIが開き、アイテムのアンロックが可能になります。

### 管理者向け：UI編集モード
管理者は、テックツリーUIを開くと画面上に **「Admin」** または **「Edit mode」** などの編集メニューが表示されます。これを使用して以下の操作が可能です。

- **ノード（アイテム）の追加・削除**: 任意のアイテムをツリーに配置できます。
- **価格（Price）の変更**: アンロックに必要なスクラップ数等のコストを変更できます。
- **通貨の変更**: 基本実装でスクラップ（ItemID: -932201673）となっていますが、独自通貨等を設定することも可能な設計です。
- **接続（パス）の編集**: どのアイテムをアンロックすれば次に進めるかを矢印で繋ぎます。
- **カスタム画像・名前の設定**: ノードのアイコンをURLから取得したり、名前や説明を多言語で独自に設定できます。
- **バニラからの除外**: 標準のツリーに含まれる特定のアイテムだけを取り除く（Exclude）ことが可能です。
- **グリッドの拡張**: `Grid Setting` からUIの広さを拡張・縮小できます。
- **Engineering（エンジニアリング）ツリー**: レベル1～3のワークベンチとは別に、`io research table` などに紐づく特殊なカテゴリのツリーを作成することもできます。

## コンソールコマンド（管理者用・補助用）
原則として**全てUI上で完結する**ように設計されていますが、UIがスタックした場合のリセットや一括操作のため、以下のコンソールコマンドが存在します。※基本的には使用しません。

| コマンド | 説明 |
|---|---|
| `techtree.cmd.open` | 自身の画面にテックツリーUIを強制的に開きます。 |
| `techtree.cmd.deleteallnodes` | 現在編集中のツリーの全てのノードを削除します。**（※警告なしで実行される可能性があります）** |
| `techtree.cmd.resetvanilla` | 現在編集中のツリーを、Rustの標準（バニラ）の状態にリセットします。**（※カスタム設定が消去されます）** |
| `techtree.cmd.excluded toggleimports` | 今後Rust本体のアップデートで追加されるバニラアイテムの、自動インポート機能のON/OFFを切り替えます。 |

## その他連携・仕様
* **Economy連携**: `Economics` などのプラグインと連携し、スクラップ以外の経済システムでツリーを進行できるサポートが含まれています。
* **CraftMenu連携**: `CraftMenu` プラグインが導入されている場合、テックツリーを開くための統合ボタンがUIに描画されます。
* **ImageLibrary不要**: 外部データベースから直接画像をキャッシュ・利用する仕組みを持っています（Carbon環境であれば `ImageDatabaseModule` を利用）。
