# xSkillSystem プラグイン 使用方法

## 概要
各種アクションに応じてXP（経験値）を獲得し、スキルレベルを上げるシステム（RPG風成長要素）を追加するプラグインです。
木こり、採掘、採集、戦闘（Slayer）から、建築、クラフト、さらにはVitality（生存・回復）まで、多様なスキルが用意されています。

## コマンド一覧（デフォルト）
これらのコマンドは設定 (`oxide/config/xSkillSystem.json`) で変更またはエイリアス（別名）を追加可能です。

| コマンド | 説明 |
|---|---|
| `/skills` | 自身のメインのスキルUIを開きます。ここから各レベルや現在XP、レベル報酬等を確認できます。 |
| `/stats` | 現在のステータス詳細（どのアイテムからどれだけXPが得られるか等）のUIを開くコマンドです。 |
| `/lb` | リーダーボード（ランキング）UIを開き、サーバー内での各スキルのトッププレイヤーを確認できます。 |

*注: UI内の各種パネルやポップアップの開閉は内部的なコンソールコマンド（`ui_rewards`など）で行われますが、通常プレイヤーは意識する必要はありません。*

## スキルの種類（カテゴリー）
- **Woodcutting**: 木材採集によるスキル
- **Mining**: 鉱石・石の採集によるスキル
- **Gathering**: 布、食べ物などフィールドの植物やアイテム拾いによるスキル
- **Looting**: 箱やドラム缶を開けた際に得られるスキル
- **Slayer**: 動物やNPC、他プレイヤーを討伐した際に得られるスキル
- **Skinning**: 動物の解体によるスキル
- **Building**: 建築パーツのアップグレードや設置によるスキル
- **Crafting**: アイテム作成によるスキル
- **Vitality**: 医療アイテム使用や体力による生存スキル

## 主な機能と設定
- **Gather Rate倍率**: スキルレベルに応じて採取量を徐々に増やす機能（デフォルトではオフ設定。Configの `EnableGatherRateBySkill` をtrueにすると有効化）。
- **Level Rewards（レベル到達報酬）**: 特定スキルの一定レベルに到達した際に、自動的あるいはUIからアイテム付与、コマンド実行、権限付与(`xpermission.use`など)の報酬を受け取ることができます。
- **ダブルXP（週末経験値2倍など）**: Configから曜日と時間を指定して「経験値2倍」イベントを自動開催できます。
- **HUDとXPポップアップ**: アクションを起こしてXPを獲得した際、画面（デフォルトは左下）にミニHUD表示とポップアップで加算XPが通知されます。

## 管理者向け権限・仕様
- `xskillsystem.powertools100%`: チェーンソーやジャックハマーといった「電動ツール」を使用した際のXP獲得率を、ペナルティなしの100%にする権限です。（設定で電動ツールによる獲得XPを60%等に下げる機能があるため）
- 管理者はチャットUI上のAdminタブや、コマンドインターフェースを利用できる場合があります（利用の際は管理者権限が必要です）。
